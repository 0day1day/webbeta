Zoo

https://zoo.mlw.re

The same way that your local Zoo collects and classifies a number of different animal species, MalwareZoo does it for the different malicious software ‘species’ seen on the wild.

A project initially developed by Manu Quintans, Ruben Espadas and Andriy Brukhovetskyy for the mlw.re community, MalwareZoo(or MZ) is an automated malware crawler with the main goal of classifying these samples and make both the analysis and the samples, available to the community of security researchers.

Its unique capabilities such as crawling, storage and pre-analysis make it different from any of the other opensource and commercial solutions.

Linux Sandbox

https://linux.huntingmalware.com

Linux Sandbox is a Cuckoo-based sandboxing system specifically modified for linux sample analysis.

Currently our Sandbox works with the following architectures:

  • ARM
  • MIPS
  • PPC
  • SPARK
  • x32
  • x64

Toaster Tracker

https://toaster.huntingmalware.com

Toaster is a Command and Control (C&C) server tracker, the information collected comes from our Cuckoo-based sandboxing system specifically modified for linux sample analysis.
